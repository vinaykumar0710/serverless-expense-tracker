1. Run the application locally or access the deployed version
2. Navigate to each page
3. Use your browser's screenshot tool or a tool like [Flameshot](https://flameshot.org/)
4. Save screenshots at **1280×800** or higher resolution
5. Replace the placeholder files in this directory
